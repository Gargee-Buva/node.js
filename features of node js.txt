2. You can run Js outside the browser with the help of node.js. Javascript can talk to native machines because of cpp
3. UI related features are being removed i.e window object , DOM etc. console.log(alert("no")) would be undefined on vscode.
4. npm init creates a package .json file , in the scripts section we can add our on scripts with which we can choose which file will be the starting point of the project
